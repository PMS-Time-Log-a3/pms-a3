Inventory App Quick Guide

This guide explains the main workflow for the mobile inventory system created by 黄丰兆 Fengzhao Huang 24832216 and 殷文涛 Wentao Yin 24833028.

Core Pages

1. What is the List page for?

The List page shows all remote inventory records and provides edit and delete buttons for each item.

2. How do I add a new item?

Open the Add tab, complete the required fields, choose the category and stock status, then submit the form.

3. How do I search for one item?

Open the Search tab and enter the unique item name. The page displays the matching record returned by the server.

Form Rules

1. Which fields are required?

Name, category, quantity, price, supplier name, stock status, and featured item must be completed. Special note is optional.

2. Why are quantity and price checked?

Both fields match integer columns in the database, so the app accepts whole numbers only.

Safety Notes

1. What should I check before deleting?

Confirm the item name and details first. Delete operations are permanent unless the server rejects the request.
